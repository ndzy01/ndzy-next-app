# Next.js 入门指南

Next.js 是一个强大的 React 框架，提供了许多开箱即用的功能。

## 主要特性

- 服务端渲染 (SSR)
- 静态站点生成 (SSG)
- 自动代码分割
- 简单的页面路由

## 安装 Next.js

```bash
npx create-next-app@latest my-app
cd my-app
npm run dev
```

## 创建页面

在 `pages` 目录下创建文件即可自动生成路由。
